- Click **Programs** on the **Main Menu**.

2\. - On the top tabs, click **Programs**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/5a2e1b0e-1f72-49b6-b5dc-55db5a8f6ba0/ascreenshot_b7905f090678493e9de8bd9af8e78bb3_text_export.jpeg)


3\. # ✅ **Create a New Program**

- Click **Create New**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/a1a1ec14-617d-4384-9eb3-4dcacc668842/ascreenshot_e7ec2c62921a422aa08a3db217d09596_text_export.jpeg)


4\. - **Name:** Enter the program name.
- **Description:** Enter information about the program.
- Click **Save**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/ce7e7623-d341-4b34-9b3a-90964b959eca/ascreenshot_1676e34e545b46d58f7855c4b01bd554_text_export.jpeg)


5\. # ✅ **Create a Session for the Program**

- Go to the **Sessions** tab.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/477c0158-3dbf-460e-a219-d812bd09aebb/ascreenshot_762d043d292d475f99969a2729449dea_text_export.jpeg)


6\. - Click **Create New**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/8b61d18e-66ab-4ae8-ab72-c958684b11ff/ascreenshot_9ca6fc84edb341f29a33143717bd7602_text_export.jpeg)


7\. - Click **Program** and select the new program that was created.

- **Name:** Name the session. Session names vary by facility. If helpful, you can use a **group number or identifier** here for programs that run multiple cycles.

- Select the **Location** for the program.

- For **Maximum Occupancy**, enter a number or leave blank if the class is infinite.

- Enter the **Start Date** (if applicable)

- Enter the **End Date** **only** if the program has an end date.

  - If the program is rolling leave the start and end date blank.

- Select the **Days of the Week** the class is taught.

  - For each day selected, enter the **Start Time** for that program.

- Select the **Instructor**.

- Click **Save**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/63d0f3a2-beb3-4949-b14d-863f765a82b9/ascreenshot_4ed7b9f5e5e74e23baf9af16994ee24d_text_export.jpeg)


8\. This screen lists all existing programs.\
• **Red = class is full**\
• **Green = class has vacancies**\
Colors update automatically as spots fill or open.\
Each card includes a quick summary of the program.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/24412427-c883-4909-82f6-d66d1db95c12/ascreenshot_9366324801aa44be97885310b7b61c78_text_export.jpeg)


9\. ✅ **Adding Inmates to Sessions**

- On the **Manage Programs** screen, click on a specific program to add inmates.
- Click the **+ (plus symbol)** next to **In Progress** to open the enrollment screen.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/ecdda1a1-7d9c-4441-ab67-c137b4e950d6/ascreenshot_21b25cd24c2c4dd4a3785c69d4a0a482_text_export.jpeg)


10\. Select the inmate from the dropdown list and enter the Start Date.

- Click **Save**.
- Repeat to add inmates in this program.
- If you have already added the inmate to the Waitlist or In Progress they will not appear in the drop down list.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/adda6766-5a33-479e-83ac-466dee82a5d0/ascreenshot_cdb3730997bd495593fc64aba4182dd5_text_export.jpeg)


11\. ✅ **Create Sessions for the Program**

- When it is time to have a session, go to **Session Meetings**.
- Click **+New Meeting**

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/b7f9b02e-311b-4422-9640-e7fb792cdaa4/ascreenshot_c5c996138ee54569ae7636c605f557c3_text_export.jpeg)


12\. - Select the session time and date.
- Click **Save**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/cf4738f9-3984-4bcf-bae5-123fa42f26f4/ascreenshot_be9e261ee123478688fddaf6ac76336c_text_export.jpeg)


13\. Additional sessions can be added using this same process. **Sessions cannot be edited once they are created,** but you may delete a session and rebuild a new one if a specific date or detail needs to be changed.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/7a38b3b9-361e-4699-8791-f9b3ff854143/screenshot_db852162cfc943c4b8a3dde6297b40eb_text_export.jpeg)


14\. ✅ **Update Attendance for a Session**

- **Select the session you want to open** to record attendance and add session notes.

- Options available:

  - **Present**

  - **Excused**

  - **Unexcused**

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/d7ee3476-a0d4-410e-8133-994b3f8973ab/ascreenshot_fd90a593ac4a4dffb483f70815c9b955_text_export.jpeg)


15\. **Notes section** – This is where staff can document the grade for that session, the step the inmate passed or failed, or any notes relevant to that session.

- If the inmate is absent, staff can record the reason for the absence and whether it was excused or unexcused.

- If an inmate is **temporarily released**, they will still appear on the roster to hold their spot for when they return.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-26/074df236-75f3-4f66-aeed-48aa2875c4b6/user_cropped_screenshot_af4a81c0f28442c780a9416872369362_text_export.jpeg)


16\. - **Attendance auto-saves.**
- The **Delete** button in this section is used to delete the specific session that is opened.
- Next to each inmate’s name, the system displays the **inmate's start date** and **current cell location** and **last note entered for that program.**
- Press **Return** to go back to this programs main page.

![](https://colony-recorder.s3.amazonaws.com/files/2025-12-09/73f49be8-dc98-46e5-b3d4-800d0a71011d/user_cropped_screenshot_dd82c290684e47849011efa2680485e6_text_export.jpeg)


17\. Basic session details are shown on the inmate’s card. If you need more detailed information, click on the inmate’s name to open their full program record.

![](https://colony-recorder.s3.amazonaws.com/files/2025-12-09/e5ae32da-cff0-43d9-a96d-a71865e48f6b/screenshot_b8abe33bed0f4560b278dc58e61564bc_text_export.jpeg)


18\. The **detailed view** displays all information for that inmate’s participation in the program.

If you see **“No Entry”** under Attendance and the **Entered By** field is blank, those sessions are scheduled for a future date, or no information has been entered yet to display.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/e157f639-1348-4b5e-9a21-e892ab8d0dd5/ascreenshot_d14cbf5823dd4cb890e8d0c984fe7319_text_export.jpeg)


19\. ### ✅ **Additional Actions Available**

- You can open the inmate’s **booking profile** by clicking **Open Person**.

- You can **print the program summary** for that inmate.

- **You can delete the participant from the program.**

  - **Important:** Deleting an inmate is **permanent** and removes them from the **entire program**, not just the session.

  - This action is **final** and should be used **with caution**, as it is **not** intended for marking an inmate as failed — it fully removes their program participation.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/cfb76750-59be-4a45-a110-d819b3cde70d/ascreenshot_c0208318debf4428b19a04159afb98cd_text_export.jpeg)


Alert: Deleting an inmate fully removes them from the program with no recovery.


20\. ✅ **Using Pass/Fail Statuses**

- Select the inmate in the program whose status needs to be updated.
- When an inmate **completes** a class **(Pass)** or is **terminated** from it **(Fail)**, this is where the status change will be made.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/ac3358eb-4451-4725-ad2b-01e3ba0f8146/ascreenshot_9971182f307f42e194a4e9bc636c7071_text_export.jpeg)


21\. - Enter the **Start Date** of the program.
- Enter the **End Date**, which is the date the inmate either completed the program or was terminated/failed.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/be32851a-0f47-4421-be8d-43b952835977/ascreenshot_66316aac2bc849ab9f66832a24f17e73_text_export.jpeg)


22\. Once the dates are entered:

- Choose the **Completion Status** that applies:

  - **Passed** = inmate **completed** the program

  - **Failed** = inmate was **terminated or removed** from the program

- Click **Update** to save the status.

Updating the status changes the inmate’s standing in the program and automatically moves their program card into the correct category based on what was selected.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/197a64da-d75e-4b0f-86a2-f584773767f2/ascreenshot_dd6f7f8f23e54f16a00cc968f1ecf43e_text_export.jpeg)


23\. When an inmate is marked as **Passed** or **Failed** and is removed from the active class they will display in the corresponding column.

![](https://colony-recorder.s3.amazonaws.com/files/2025-12-09/687df972-abf2-47b4-a980-16f25a4bd06b/user_cropped_screenshot_067f333f198a423faf807c7942a64e05_text_export.jpeg)


24\. The booking status displays on the tag next to the inmate's information as ACTIVE or RELEASED.

![](https://colony-recorder.s3.amazonaws.com/files/2025-12-09/0876b2b6-a7c1-44c2-8627-afc560339b63/user_cropped_screenshot_2b1b6451ee584f6a9474272cff037af5_text_export.jpeg)


25\. ✅ **Quick View of an Inmate’s Program Information**

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/c7f4867b-d7e3-4463-962f-6916f08aa591/ascreenshot_b9230ea656ab41389f30896ea918715d_text_export.jpeg)


26\. - Select the inmate and click **Open Person**.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/45c68665-48ad-4ce4-92fe-bfc329bc91ea/ascreenshot_e5b4fdf4385248fe85a8b210418a948e_text_export.jpeg)

28\. Scroll to the bottom of the booking page.

- Two sections will display:
  - **Left side:** A list of all programs the inmate has been associated with.

  - **Right side:** Color-coded attendance notes for each class session.

![](https://colony-recorder.s3.amazonaws.com/files/2025-11-25/1f1cec9b-acf6-4b25-9ab9-5bd5e5d22f3f/ascreenshot_c8cf60e7493a428393a5fa5c794274c3_text_export.jpeg)


29\. ✅**Printing Program Information**

- Click **“Print Only From Booking”** to print the inmate’s program information.
- Click **“Print All for Person”** to print all historic programs associated with the inmate.
